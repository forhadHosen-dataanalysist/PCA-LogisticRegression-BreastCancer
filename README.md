# PCA-LogisticRegression-BreastCancer
A machine learning project for early breast cancer detection using PCA for feature reduction and Logistic Regression for classification.

## Dataset
The dataset contains clinical and biological features to distinguish between benign and malignant tumors.

## Repository Structure
- `data/` : Contains the dataset.
- `notebooks/` : google collab with EDA, PCA, and model implementation.
- `src/` : Python scripts for preprocessing, PCA, model training, and evaluation.
- `report/` : Final report document.
- `requirements.txt` : Required Python packages.
